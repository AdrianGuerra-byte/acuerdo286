# Portal Informativo - Acuerdo 286

Esta página web es un portal informativo dedicado al proceso de titulación mediante el Acuerdo 286, diseñado para facilitar a los usuarios el acceso a toda la información necesaria para obtener su certificación.

El sitio ofrece detalles sobre los requisitos, fechas de examen, el proceso de inscripción y material de estudio. El objetivo es centralizar y simplificar la consulta de información para validar conocimientos y habilidades a nivel profesional.

## Stack de Tecnología

- **Framework:** [Next.js](https://nextjs.org/)
- **Lenguaje:** [TypeScript](https://www.typescriptlang.org/)
- **UI:** [React](https://react.dev/)
- **Componentes:** [Shadcn/ui](https://ui.shadcn.com/)
- **Estilos:** [Tailwind CSS](https://tailwindcss.com/)
- **Gestor de Paquetes:** [pnpm](https://pnpm.io/)
- **Despliegue:** [Vercel](https://vercel.com/)
